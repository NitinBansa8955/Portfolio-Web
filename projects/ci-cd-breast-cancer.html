CI/CD Pipeline for Breast-Cancer Classification with MLRun

Advisor: Prof. Mayank Singh, IIT Gandhinagar

Overview

Implemented an automated MLRun pipeline to handle data ingestion, hyperparameter tuning, model training and deployment for breast-cancer classification.

Results

Random Forest trained on scikit-learn breast-cancer dataset achieved 96.49% test accuracy with strong precision.

Pipeline

  • Data ingestion and preprocessing steps automated.
  • Hyperparameter tuning using grid/random search.
  • Model packaging and deployment using MLRun.
